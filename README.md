## Requires 📃

- pygame

```cmd
pip install pygame
```

- pyinstaller (optional) for build

```cmd
pip install pyinstaller
```

## How to run 🏃‍♀️

```cmd
py main.py
```

or build file .exe

```cmd
pyinstaller --onefile --windowed --icon=app.ico main.py
```
